# Система учёта домашних обязанностей и ротации задач в семье

Курсовая работа: **Проектирование и разработка интерфейса** системы учёта домашних обязанностей и ротации задач в семье.

## Технологии

- **React 19** + **Vite 7**
- **React Router** — навигация
- **localStorage** — хранение данных (члены семьи, задачи, настройки)

## Запуск

```bash
npm install
npm run dev
```

Откройте в браузере адрес, указанный в терминале (обычно http://localhost:5173).

## Разделы приложения

| Раздел | Назначение |
|--------|------------|
| **Домашняя панель** | Сводка по семье: прогресс по участникам, срочные задачи, «Лучший помощник», перераспределение исполнителей |
| **Задачи** | Создание и редактирование задач, список/доска, фильтры по участнику и статусу, отметка «Выполнено» с ротацией, восстановление из архива |
| **Календарь** | Просмотр нагрузки по дням (день/неделя/месяц), повторяющиеся задачи, фильтр по участнику |
| **Профиль** | Выбор участника, прогресс, баллы по сложности, награды за 100/200 баллов, статистика по дням недели |
| **Настройки** | Добавление/удаление членов семьи (с подтверждением, если есть задачи), награды за баллы |

## Реализованный функционал

- Учёт членов семьи (добавление/удаление; при удалении — предупреждение, если у участника есть задачи)
- Создание и **редактирование** задач (название, описание, участники, срок, повтор, сложность)
- Ротация исполнителя при нажатии «Выполнено» (следующий по кругу)
- **Перераспределение** — модальное окно для смены текущего исполнителя по каждой задаче
- Подсветка **просроченных** задач (бейдж и стиль карточки)
- **Восстановление** задачи из архива (кнопка «Вернуть в активные» при фильтре «Завершённые»)
- Баллы по сложности (лёгкая — 5, средняя — 10, сложная — 15); награды за 100 и 200 баллов в Настройках и Профиле
- Ссылки «Мои задачи» и «Мои задачи сегодня» ведут на раздел «Задачи» с фильтром по выбранному участнику
- Статистика в Профиле по дням недели
- Календарь: повторяющиеся задачи (каждый день / по выходным), задачи без срока вынесены отдельно

## Структура проекта

```
src/
  components/   — AddTaskForm, TaskCard, EditTaskModal, RedistributeModal
  constants/    — storage (ключи localStorage)
  context/      — FamilyContext (члены семьи), TasksContext (задачи и действия)
  pages/        — Home, Dashboard, Calendar, Profile, Settings
  utils/        — rotationUtils, dateUtils (даты/просрочка/доска), taskUtils (баллы, счётчик задач)
```

Данные сохраняются в `localStorage` (ключи: `familyMembers`, `tasks`, `rotationSettings`).
