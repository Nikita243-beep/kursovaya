# 🏠 Система учёта домашних обязанностей и ротации задач в семье

Веб-приложение для планирования домашних дел, равномерного распределения обязанностей между членами семьи и отслеживания прогресса с системой баллов и наград.

## 📖 О проекте

Приложение помогает семье:

- вести общий список домашних задач с дедлайнами и повторениями;
- автоматически назначать исполнителя по кругу (ротация) при отметке «Выполнено»;
- перераспределять задачи вручную при необходимости;
- начислять баллы за сложность и отображать награды за достижения (100, 200+ баллов);
- просматривать нагрузку по дням в календаре и статистику по участникам в профиле.

Данные хранятся локально в браузере (`localStorage`), без сервера и регистрации.

## 🚀 Быстрый старт

### Требования

- [Node.js](https://nodejs.org/) (рекомендуется LTS, например 18+)
- npm (идёт в комплекте с Node.js)

### Установка и запуск

```bash
# Клонировать репозиторий (или перейти в папку проекта)
cd SistemHomeTrecker

# Установить зависимости
npm install

# Запустить режим разработки
npm run dev
```

В терминале появится адрес вида `http://localhost:5173` — откройте его в браузере.

## 📱 Разделы приложения

| Раздел | Описание |
|--------|----------|
| **Домашняя панель** | Сводка по семье: прогресс участников, срочные задачи, «Лучший помощник недели», кнопка перераспределения исполнителей. |
| **Задачи** | Создание и редактирование задач, вид «список» / «доска», фильтры по участнику и статусу, отметка «Выполнено» с ротацией, восстановление из архива. |

| **Настройки** | Добавление и удаление членов семьи (с предупреждением при наличии задач), настройка наград за баллы. |

## ✨ Функциональность

### Семья и участники

- Добавление и удаление членов семьи.
- При удалении участника — предупреждение, если у него есть назначенные задачи.

### Задачи

- Создание и редактирование: название, описание, участники ротации, срок, повтор (каждый день / по выходным / без повтора), сложность (лёгкая / средняя / сложная).
- Ротация: при нажатии «Выполнено» следующий исполнитель назначается по кругу.
- Перераспределение: модальное окно для смены текущего исполнителя по каждой задаче.
- Подсветка просроченных задач (бейдж «Просрочено» и стиль карточки).
- Восстановление из архива: кнопка «Вернуть в активные» при фильтре «Завершённые».

### Баллы и награды

- Баллы за сложность: лёгкая — 5, средняя — 10, сложная — 15.
- Награды за 100 и 200 баллов настраиваются в **Настройках** и отображаются в **Профиле**.

### Навигация и фильтры

- Ссылки «Мои задачи» и «Мои задачи сегодня» ведут в раздел «Задачи» с фильтром по выбранному участнику.
- В Профиле — статистика по дням недели.
- В Календаре — повторяющиеся задачи и отдельный блок для задач без срока.

## 💾 Хранение данных

Все данные хранятся в **localStorage** браузера:

| Ключ | Содержимое |
|------|------------|
| `familyMembers` | Список членов семьи |
| `tasks` | Задачи (активные и завершённые) |
| `rotationSettings` | Настройки ротации и наград |

Сервер и база данных не используются. Очистка данных браузера удалит и данные приложения.

## 📐 Адаптивность

Интерфейс адаптирован под разные размеры экрана:

- **До 400px** — узкие телефоны: компактные отступы, формы в одну колонку.
- **До 600px** — телефоны: навигация в колонку, доска задач в одну колонку, календарь и профиль перестраиваются.
- **601–768px** — планшеты: доска задач в 2 колонки, сетка участников в 2 колонки.
- **От 921px** — десктоп: увеличенные отступы, hover-эффекты на карточках.
- **От 1200px** — широкая область контента до 1000px для удобного чтения.
